# v1.0.0

Initial release of Task Journal.

## Features

- **Calendar view** — Monthly grid with task titles. Navigate months, tap a day to see and manage its tasks.
- **Task management** — Create, edit, and delete tasks with title and notes. Tasks are stored per day.
- **All Tasks** — Browse all tasks grouped by date.
- **Repeated Tasks** — Create task templates (e.g., "Standup", "Gym") for quick re-use. See when each was last logged ("3 days ago") and view full history.
- **Categories** — Organize repeated tasks into categories. Add, select, and delete categories from the template editor.
- **Template picker** — When creating a new task from the Journal, search and select a repeated task template to pre-fill the form.
- **Settings** — First day of week (Sunday/Monday).
- **LAN Sync** — Sync tasks, repeated tasks, and categories between devices over local network. Timestamp-based incremental sync with server-wins-on-tie conflict resolution.
- **Dark mode** — Automatic light/dark theme matching system preference.
- **Sync server** — Standalone Go server for LAN sync. Docker image available at `ghcr.io`.

## Tech

- Expo SDK 56 (React Native 0.85, React 19)
- SQLite via `expo-sqlite` with UUIDv7 string IDs, Unix ms timestamps, soft deletes
- React Compiler for automatic memoization
- `expo-symbols` for cross-platform icons
- Go sync server with WASM-based SQLite (no CGO)
